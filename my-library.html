MyLibrary Cataloging App

A simple app that catalogs a user"s book collection. It captures notes, cover image, and a rating for each book. The catalog is stored on the user"s device using localStorage and is available on or offline. Built with HTML5, Sass and JavaScript.

Live Demo | GitHub Source

localStorage

We used the built in localStorage feature of HTML5 to get a rough protoype up and running quickly. It simulates the expereince of using a server-side database without having to spend the time setting one up.
