Rapid HTML Prototyping

As part of the Premera UX team, I built interactive prototypes of new site features. Prior to my role, designs were created with standard graphics software and weren't a true reflection of the end product. Using HTML and the same CSS framework used in production allowed us to test, refine, and deliver designs that were easy for developers to implement.
